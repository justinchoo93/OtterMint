# Repo Skills

## Available skills

- `security-audit`: Use when the user asks for a security audit, security hardening, Supabase security finding remediation, auth/session security review, or security policy/documentation updates. File: `.claude/skills/security-audit/SKILL.md`
- `file-issue`: Use when the user wants to file, create, or open a GitHub issue for a bug, feature, improvement, or task. Generates structured acceptance criteria. File: `.claude/skills/file-issue/SKILL.md`

## Usage notes

- Open the referenced `SKILL.md` and follow it before starting substantial work.
- Keep outcomes concrete: findings, code fixes, verification, and documentation updates.
